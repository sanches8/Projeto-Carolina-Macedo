# 🌟 Projeto Carolina Macedo - Site Institucional

Este é um projeto de site desenvolvido para a **EMEF Professora Carolina de Moraes Macedo**, com o objetivo de apresentar informações relevantes da escola de forma moderna e acessível para pais, alunos e comunidade em geral.

---

## 📌 Funcionalidades

- ✅ Banner institucional com imagem personalizada
- ✅ Menu de navegação responsivo (com ícone hamburguer para mobile)
- ✅ Seção "Sobre a Escola"
- ✅ Lista de projetos desenvolvidos
- ✅ Mapa com localização da unidade escolar
- ✅ Rodapé com informações de contato e links úteis

---

## 💻 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Font Awesome (ícones)
- Google Maps Embed

---

## 📱 Responsividade

O site foi desenvolvido com foco na experiência mobile, utilizando técnicas de flexbox, media queries e menus adaptativos para garantir boa navegação em diferentes dispositivos.

---

## 🚀 Como usar

1. Faça o clone do repositório:

```bash
git clone https://github.com/sanches8/Projeto-Carolina-Macedo.git
```
2. Abra o arquivo index.html em seu navegador.
